Architecture: Used Next.js app router for a modern foundation.

Dependencies: Leveraged react-day-picker to ensure robust, accessible date-math logic, allowing me to focus entirely on UX, styling, and state management.

State: Used React state synced with localStorage. The notes are dynamically tied to the start date of any selected range.

Styling: Used Tailwind CSS for a fluid, mobile-first design, alongside framer-motion for micro-interactions (fade-ins, text replacement).
